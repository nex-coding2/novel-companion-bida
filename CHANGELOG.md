# Changelog

## [0.2.0] - 2026-07-23

### Added - R0 多书籍管理与创建引导
- 作品架多项目切换（扫描全部 novel.yaml、卡片式、当前活跃书高亮、最近书籍记忆、空态引导）
- 新建书籍 Webview 向导（题材模板预览、目录预览、一键创建+打开）
- 新建卷命令、新建章节向导增强（章节号预览、卷选择、默认标题）
- 空态欢迎与新手引导、内嵌帮助

### Added - R1 长篇工程加固
- 伏笔深度集成（写作时未回收提示 CodeLens、归档前检查、引用反查、上下文打包）
- 时间线深度集成（story_time 汇总、前序摘要打包、归档生成）
- 卷管理（重命名/删除/扁平迁移）、章节树拖拽排序
- 全书跨章查找替换（含改名联动、预览撤销）
- 导出 Markdown/TXT

### Added - R2 创作辅助与数据安全
- 起名器（本地姓+字库、Webview 面板、AI 起名任务、写入人物卡）
- 语料片段库（项目级 snippets、树视图、搜索插入）
- 续写建议（草稿缓冲、对照采纳、权限限定）
- 本地快照历史（保存时快照、对比回滚、Git 协同）
- 软删除回收站（恢复/彻底删除/过期清理）
- Git 工作流向导（提交向导、历史/Diff、卷/里程碑标签）
- 写作日历（热力图、连续日更、目标达成率）

### Added - R3 一致性与通读
- 全书一致性检查（伏笔/时间线/人物/设定/战力五类，报告面板）
- 通读视图（只读合并、批注笔记）
- 冲突检测（伏笔遗漏、时间线矛盾）

### Added - R4 沉浸与可视化
- 沉浸写作视图（纸张/字体/行距/夜间/聚焦、防抖自动保存、主题持久化）
- 人物关系图（力导向、节点跳转、拖拽）
- 时间线可视化（横轴章节×纵轴人物泳道、冲突高亮）

### Added - R5 扩展与远期
- EPUB/DOCX 排版导出（原生 ZipWriter，无外部依赖）
- MCP Server（只读工具集，安装命令，Kilo/Cline/Continue 配置）
- 本地加密保险箱（AES-256-GCM，口令不落盘）
- AI 直连（可选，OpenAI 兼容端点，结果走审阅面板）

### Fixed
- EPUB 卷标识改用索引避免卷名碰撞损坏；XML 屇性转义防注入
- DOCX markdown 渲染统一识别标题层级与加粗
- AI 直连 taskId 唯一化避免覆盖已有审稿结果
- 加密保险箱锁顺序修正避免明文残留；scrypt 异步不阻塞主线程
- MCP Server 符号链接越界读取防护
